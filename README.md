# Calculator App

간단한 사칙연산 계산기 프로그램입니다.  
정수와 실수를 모두 입력할 수 있으며, 음수 입력은 제한됩니다.  
사용자가 `exit`을 입력하기 전까지 반복해서 계산할 수 있습니다.

## 기능
- 덧셈, 뺄셈, 곱셈, 나눗셈 지원
- 정수/실수 입력 가능
- 음수 입력 제한 (0 포함 양수만 허용)
- 결과 기록 저장 및 삭제 기능
- `exit` 입력 전까지 반복 실행

## 실행 방법
1. 프로그램 실행
2. 첫 번째 숫자 입력
3. 두 번째 숫자 입력
4. 연산자 입력 (+, -, *, /)
5. 결과 확인
6. `exit` 입력 시 종료
